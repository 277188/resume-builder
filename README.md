# 简历生成器

一个现代化的在线简历生成工具，帮助您快速创建专业、美观的简历。支持多种模板选择、实时预览和便捷的PDF导出功能。

## 主要特性

### 多样化模板选择
- **9种专业模板**：任您选择
- **实时模板切换**：即时预览不同模板效果
- **响应式设计**：在各种设备上都能完美显示

### 强大的编辑功能
- **拖拽排序**：轻松调整简历各板块顺序
- **实时预览**：编辑内容即时同步显示在预览区域
- **自定义板块**：灵活添加、删除和重命名简历板块
- **智能表单**：根据内容类型自动匹配最合适的输入字段

### 优质用户体验
- **直观界面**：清晰的三栏式布局，操作简单明了
- **键盘快捷键**：支持Ctrl+P快速打印
- **缩放控制**：灵活调整预览区域大小
- **自动保存**：数据自动保存至本地存储，防止丢失

### 便捷的导出与分享
- **PDF导出**：一键生成高质量PDF简历
- **打印优化**：专门的打印样式，确保输出效果完美
- **居中布局**：PDF导出自动居中排版，呈现专业美观效果

## 在线访问

## 使用指南

1. **选择模板**：从顶部选择喜欢的简历模板
2. **编辑内容**：在右侧面板填写个人信息、教育背景、工作经历等
3. **管理板块**：在左侧通过拖拽调整板块顺序，或添加自定义板块
4. **实时预览**：中间区域实时显示简历效果
5. **导出PDF**：点击保存按钮即可生成PDF文件

## 技术架构

- **前端技术**：纯HTML5、CSS3、JavaScript (ES6+)
- **PDF生成**：html2pdf.js
- **拖拽功能**：Sortable.js
- **响应式适配**：CSS媒体查询

## 响应式设计
- **桌面端**：完整三栏布局，功能齐全
- **平板端**：自适应调整，保持良好可用性
- **移动端**：单列布局（未作优化，推荐电脑运行）
